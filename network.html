网络诊断工具

网络连接测试、端口扫描、网络配置查看等网络诊断工具集合，支持 Windows 和 Linux 系统

端口占用检查

Linux 端口检查

查看所有监听端口

netstat -tlnp

查看所有端口连接

netstat -an

查看特定端口占用

netstat -tlnp | grep :8080

使用 ss 命令查看端口

ss -tlnp

查看特定端口的进程

lsof -i :8080

查看某个进程的端口占用

lsof -p [PID]

Windows 端口检查

查看所有端口连接

netstat -an

查看端口和进程信息

netstat -ano

查看特定端口占用

netstat -ano | findstr :8080

查看监听端口

netstat -an | findstr LISTENING

根据PID查看进程详情

tasklist /fi "PID eq [PID]"

终止占用端口的进程

taskkill /PID [PID] /F

网络连通性测试

基础连通性测试

Ping 测试

ping google.com

限制 Ping 次数 (Linux)

ping -c 4 google.com

限制 Ping 次数 (Windows)

ping -n 4 google.com

测试本地回环

ping 127.0.0.1

高级连通性测试

Traceroute 路由追踪 (Linux)

traceroute google.com

Tracert 路由追踪 (Windows)

tracert google.com

测试特定端口连通性

telnet google.com 80

使用 nc 测试端口

nc -zv google.com 80

网络配置查看

Linux 网络配置

查看网络接口

ip addr show

查看网络接口 (旧命令)

ifconfig

查看路由表

ip route show

查看路由表 (旧命令)

route -n

查看 DNS 配置

cat /etc/resolv.conf

Windows 网络配置

查看网络配置

ipconfig

查看详细网络配置

ipconfig /all

查看路由表

route print

刷新 DNS

ipconfig /flushdns

释放 IP 地址

ipconfig /release

重新获取 IP 地址

ipconfig /renew

DNS 诊断工具

DNS 查询

DNS 解析查询

nslookup google.com

使用 dig 查询 DNS

dig google.com

查询 MX 记录

dig google.com MX

反向 DNS 查询

dig -x 8.8.8.8

高级 DNS 测试

指定 DNS 服务器查询

nslookup google.com 8.8.8.8

查询 DNS 传播

dig +trace google.com

查询 TXT 记录

dig google.com TXT

查询所有记录

dig google.com ANY

网络监控和流量分析

流量监控

实时网络流量监控

iftop

网络接口统计

cat /proc/net/dev

实时带宽监控

nload

网络连接统计

ss -s

高级网络工具

网络扫描 (nmap)

nmap -sn 192.168.1.0/24

端口扫描

nmap -p 1-1000 192.168.1.1

HTTP 响应测试

curl -I https://google.com

网络性能测试

wget -O /dev/null http://speedtest.wdc01.softlayer.com/downloads/test10.zip

网络安全诊断

防火墙检查

查看 iptables 规则

sudo iptables -L -n

查看 ufw 状态

sudo ufw status

检查开放端口 (nmap)

nmap localhost

Windows 防火墙状态

netsh advfirewall show allprofiles

连接安全检查

检查可疑连接

netstat -an | grep ESTABLISHED

查看活跃连接数

ss -s

SSL/TLS 连接测试

openssl s_client -connect google.com:443

证书信息查看

echo | openssl s_client -servername google.com -connect google.com:443 2>/dev/null | openssl x509 -noout -dates
