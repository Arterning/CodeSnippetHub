Git 命令大全

Git版本控制系统常用命令，涵盖仓库管理、分支操作、远程协作等场景

基础操作

初始化仓库

在当前目录创建新的Git仓库

命令:
git init

克隆仓库

从远程仓库克隆到本地

命令:
git clone <repository-url>

查看状态

查看工作区和暂存区状态

命令:
git status

添加文件

将文件添加到暂存区

常用命令:
git add . # 添加所有文件
git add filename # 添加指定文件

提交更改

将暂存区内容提交到仓库

命令:
git commit -m "commit message"

查看日志

查看提交历史

常用命令:
git log --oneline # 简洁格式
git log --graph # 图形化显示

分支管理

查看分支

查看本地和远程分支

命令:
git branch -a # 查看所有分支
git branch -r # 查看远程分支

创建分支

创建新分支

命令:
git branch <branch-name>

切换分支

切换到指定分支

命令:
git checkout <branch-name>

创建并切换分支

一次性创建并切换到新分支

命令:
git checkout -b <branch-name>

合并分支

将指定分支合并到当前分支

命令:
git merge <branch-name>

删除分支

删除本地分支

命令:
git branch -d <branch-name> # 安全删除
git branch -D <branch-name> # 强制删除

远程仓库

查看远程仓库

查看已配置的远程仓库

命令:
git remote -v

添加远程仓库

添加新的远程仓库

命令:
git remote add <name> <url>

推送到远程

将本地更改推送到远程仓库

命令:
git push <remote> <branch>

拉取更新

从远程仓库拉取最新更改

常用命令:
git pull # 拉取并合并
git fetch # 仅拉取不合并

高级操作

暂存更改

临时保存当前工作区更改

常用命令:
git stash # 暂存更改
git stash pop # 恢复暂存
git stash list # 查看暂存列表

重写历史

修改提交历史

常用命令:
git commit --amend # 修改最后一次提交
git reset --soft HEAD~1 # 撤销最后一次提交但保留更改

标签管理

创建和管理版本标签

命令:
git tag <tag-name> # 创建标签
git push origin <tag-name> # 推送标签

查找和比较

查找提交和比较差异

常用命令:
git diff # 查看工作区差异
git diff --staged # 查看暂存区差异
git blame filename # 查看文件修改历史
